**🍥 Nindo Chronicles — Naruto Shippuden Web Zine**
## 🎨 UI Design Hackathon Project | Frontend-Only Web Experience


> An immersive single-page web experience designed as a cinematic Naruto Shippuden digital zine featuring animated storytelling, interactive UI components, and multimedia elements.


---

---

# 🎨 Hackathon Context

This project was developed for a **UI Design Hackathon Challenge** where only the **frontend interface** was required.

⚠️ No backend functionality was implemented.

The focus of the challenge was to demonstrate **creative frontend engineering, animation systems, and immersive storytelling design.**



---

**👨‍💻 Role**
### Solo Developer

Responsible for:

        • Complete frontend layout architecture  

        • Animation system implementation  

        • Multimedia interaction logic  

        • Responsive UI component design  

        • Scroll-based storytelling experience  



---

**🧠 Problem Statement**

Traditional static web pages struggle to deliver engaging storytelling experiences for entertainment content.

The challenge was to design a visually immersive interface capable of delivering:

        • Cinematic presentation  

        • Interactive character exploration  

        • Dynamic storytelling sections  

        • Smooth animated transitions  


All implemented within a **frontend-only environment.**



---

**🚀 Solution**

Built an interactive **single-page web zine experience** combining cinematic design and interactive UI elements.

The experience includes:

        • Video backgrounds  

        • Animated UI components  

        • Character interaction cards  

        • Voice-line playback  

        • Timeline storytelling  

        • Scroll-based animations  


The interface guides users through Naruto’s journey using cinematic presentation and interactive components.



---

**🔧 Core Features**


### 🎬 Cinematic Intro Experience

        • Fullscreen intro overlay  

        • Click-to-start cinematic intro video  

        • Loading spinner during media initialization  

        • Smooth fade transition into main interface  

        • Background music activation after intro  



### 🌅 Hero Section

        • Looping hero background video  

        • Dark overlay for visual readability  

        • Japanese-inspired typography styling  

        • Animated gradient title  

        • Scroll indicator animation  



### 🥷 Shinobi Character Gallery

Interactive flip-card gallery featuring major Naruto characters.

Each card includes:

Front Side

        • Character image  

        • Character name  


Back Side

        • Character quote  

        • Short biography  

        • Voice-line playback  


Additional Logic

        • Only one voice line can play at a time  

        • Audio automatically stops when the card flips back  



### ☁️ Akatsuki Horizontal Gallery

Drag-to-scroll gallery for Akatsuki members.

Features include:

        • Horizontal character card strip  

        • Cursor interaction feedback  

        • Character rank and role display  

        • Smooth drag scrolling interaction  



### 📜 Legendary Arcs Timeline

Interactive vertical timeline showcasing key Naruto story arcs.

Included arcs:

        • Kazekage Rescue  

        • Sasuke Retrieval  

        • Pain's Assault  

        • Fourth Ninja War  

        • Final Valley Battle  


Clicking a timeline node opens a popup containing:

        • Arc image  

        • Arc title  

        • Description  

        • Close button and click-outside closing  

        • Image fallback handling  



### 🗣 Iconic Quotes Section

Dynamic quote cards featuring iconic dialogue.

Implemented using:

        • IntersectionObserver  

        • Custom typewriter animation  

        • Scroll-triggered rendering  


Quotes appear **character-by-character** when the section enters the viewport.



### 🎞 Scroll-Based Animations

UI animations triggered using the **AOS animation library** including:

        • Fade-up transitions  

        • Card flip animations  

        • Section reveal effects  



---

**🛠 Tech Stack**

Frontend Technologies

        • HTML5  

        • CSS3 (custom properties, animations, gradients)  

        • Vanilla JavaScript  


Libraries

        • AOS (Animate On Scroll)  

        • Font Awesome  

        • Google Fonts  


Media Assets

        • Background video sections  

        • Character voice audio  

        • Animated GIF ending  



---

**🏗 System Architecture**

The interface is structured through layered frontend interaction systems.

        1️⃣ Intro Overlay System  

        2️⃣ Media Playback Control Layer  

        3️⃣ Interactive UI Component Layer  

        4️⃣ Scroll Animation Layer  

        5️⃣ Multimedia Storytelling Sections  


All logic implemented using **pure frontend scripting.**
