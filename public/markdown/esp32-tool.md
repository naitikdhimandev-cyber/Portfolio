# ESP32 Cybersecurity Testing Tool

A comprehensive IoT-based offensive security tool designed for network auditing and ethical testing.

## Capabilities
- **Honeypot Simulation**: Deploy decoy systems to trap attackers.
- **Evil Twin Testing**: Audit WiFi security by simulating access points.
- **IR Cloner & Jammer**: Test signal security for physical devices.
- **Network Scanning**: Comprehensive audit of local network vulnerabilities.
- **SSID Cloning**: MITM testing for educational and audit purposes.

## Hardware
- **Core**: ESP32 Microcontroller.
- **Interface**: Custom C++/Python scripts for network orchestration.
